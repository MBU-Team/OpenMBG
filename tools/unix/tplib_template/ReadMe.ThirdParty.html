  • The OpenAL binary (libopenal.so) provided with this distribution is built from an OpenAL snapshot. That snapshot can be downloaded from here: http://www.garagegames.com/docs/torque.sdk/gstarted/openal.html. For more information about OpenAL check out opensource.creative.com and www.openal.org.
  • The GNU C/C++ library used is version XXXXX. Check http://www.fsf.org for more information about these libraries.
  • The GLU library is from verson XXXXX of the Mesa 3D graphics library. This library is statically linked for compatibility. http://www.mesa3d.org
